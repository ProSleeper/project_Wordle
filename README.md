## 프로젝트 명
### Wordle (단어맞추기 게임)


## 프로젝트 개요
 - 5개의 알파벳으로 이루어진 단어를 맞추는 게임으로, 
    6번의 시도 내에 맞춰야 하고 알파벳만 맞추면
    노란색, 알파벳과 위치를 모두 맞추면 녹색으로 표시
    매일 자정에 새로운 단어로 교체 됩니다.
    

## 개발 환경
- HTML, CSS, Javascript, VSCode, Java 11, SpringBoot 2.7.3
- 참여인원: 1인 프로젝트
- 개발기간: 022.07.16 ~ 2022.07.25


## 실행 환경
- Web, Mobile Web
